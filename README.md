# ElectroHub - Electronics Website

## Project Overview
MyStore is an e-commerce website for electronics where users can browse, view, and explore products.

## Setup & Installation
1. Clone the repo:  
2. Install dependencies:
   npm install
3. Start development server:
   npm run dev
4. Open in browser: `http://localhost:3000`
## Routes

* `/` → Home
* `/products` → Products List
* `/products/:id` → Product Detail
* `/about` → About Page
* `/login` → Login Page

## Features

* Browse products
* View product details & related products
* Most Popular products filter
* Login/Logout with cookies
* Responsive design (mobile & desktop)
